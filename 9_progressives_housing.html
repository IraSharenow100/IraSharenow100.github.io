El Cerrito's housing pipeline has nearly stalled.
The state requires El Cerrito to permit 1,391 housing units — including 588 very-low-income and 339 low-income units — between 2023 and 2031. Through 2024, roughly 98 units have been permitted. The El Cerrito Plaza TOD is the city's only realistic path to meeting that obligation. It includes 743 units, with 351 below-market-rate. Any further delay to the TOD threatens the entire pipeline.

The library and 79 affordable units share the same uncertain grant.
The library is planned for the ground floor of Parcel C West — the same building that would house 79 affordable rental units. Both the library and those housing units depend on the developer winning an AHSC state grant. The developer has never applied for this grant on Parcel C West. On the adjacent Parcel C East — 84 affordable units, same developer — the application was rejected in Round 9 (2024). If the grant fails again, both the library and those 79 homes don't get built.

The grant track record at this site should alarm housing advocates.
Parcel A South — the first affordable housing phase at El Cerrito Plaza — applied for AHSC twice before winning in Round 8. Parcel C East applied in Round 9 and was rejected. Parcel C West, the library site, has never applied. A first-time application for a complex TOD project at a site with no prior approval history is a high-risk bet — especially as Cap-and-Invest auction revenues, which fund AHSC, fell sharply in 2025 — some quarterly auctions down more than 35% from the prior year — putting AHSC funding under sustained pressure.
Passing Measure C does not help the housing get built
The parcel tax funds the library's construction and operating costs. It does not contribute to the AHSC application score. It does not fund the affordable housing units. It does not accelerate the grant timeline. Voting Yes on Measure C commits residents to 30 years of taxes while doing nothing to address the actual barrier — the missing state grant — that is blocking both the library and the housing.
Bay Point — same county, same BART TOD model — secured $9.9M in state grants and $2M in federal grants for its new library. El Cerrito never applied for those programs. A housing advocate should ask: why is El Cerrito asking residents to fund 100% of library costs when peer cities are securing millions in outside funding?
