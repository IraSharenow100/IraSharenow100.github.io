2
You pay for six years before the library opens.
The parcel tax starts in December 2026. A typical El Cerrito home pays $289 in Year 1, rising every year. By the time the library opens in 2032, a typical household will have paid roughly $1,800 — for a building that doesn't exist yet. That money cannot be recovered if the project is delayed further or never built.

3
The library may never be built.
Construction is contingent on a private developer winning a competitive state housing grant — one they have never applied for on this site. The same developer applied for a nearby parcel and was rejected in the most recent round. If the grant fails, the taxes are collected and the library doesn't get built. There is no backup plan and no refund.
The current library is open now — and serves your kids today
The existing El Cerrito Library is open 46 hours per week. After school, it is already packed with children doing homework and using its resources. A No vote does not close the library. The current library stays open. A No vote rejects a risky, expensive plan — not library services themselves.
